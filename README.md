# Password generator
A random password generator made with Vuejs where you can choose its length and copy it to clipboard.

## Project setup

```npm install```

### Compiles and hot-reloads for development

```npm run serve```

### Compiles and minifies for production

```npm run build```

### Lints and fixes files

```npm run lint```
